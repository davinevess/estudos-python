# Sistema de Gerenciamento de Notas Acadêmicas

Este é um sistema em terminal desenvolvido em Python para o registro, acompanhamento e análise do desempenho acadêmico de estudantes. O projeto foi estruturado utilizando funções de responsabilidade única (SRP), documentação interna baseada na PEP 8 e tratamento de casos extremos (*edge cases*).

## 🚀 Funcionalidades

- Armazenamento estruturado de estudantes utilizando listas e dicionários.
- Cálculo automático de médias aritméticas com prevenção contra erros de divisão por zero.
- Validação de situação institucional ('Aprovado' ou 'Reprovado') com nota de corte flexível.
- Geração de relatórios de desempenho formatados em colunas diretamente no terminal.

## 📋 Pré-requisitos

Para executar este sistema e rodar os testes unitários, você precisará apenas do ambiente de execução do Python instalado em sua máquina:

- **Python 3.10 ou superior**
