# 🚀 Estudos Python - DataCode Solutions

Repositório dedicado aos meus primeiros passos na programação com Python, focado no desenvolvimento de lógica, manipulação de dados e automação de cálculos fundamentais.

## 🛠️ Desafios Realizados

### 1. Conversor de Temperatura (`conversor_temperatura.py`)
* **Objetivo:** Converter graus Celsius para Fahrenheit.
* **Conceitos:** Entrada de dados (`input`), manipulação de números decimais (`float`) e formatação de saída.

### 2. Cálculo Fatorial (`calculo_fatorial.py`)
* **Objetivo:** Calcular o fatorial de um número inteiro entre 1 e 10.
* **Conceitos:** Importação de bibliotecas nativas (`math`), tratamento de números inteiros (`int`) e aplicação de **Casting** para conversão de tipos.

## 🧠 Competências Desenvolvidas
* **Casting e Conversão:** Entendimento profundo sobre a transformação de strings em tipos numéricos.
* **I/O e Bibliotecas Nativas:** Aptidão em recolher informações e utilizar métodos prontos para otimizar o código.
* **Boas Práticas:** Documentação de código através de comentários explicativos para facilitar a manutenção.
